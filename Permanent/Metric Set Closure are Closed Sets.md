# Closure of a Set is Closed

## Context

## Statements

**Theorem** ([[A Short Introduction to Metric Spaces|@Eck23, Theorem 1.5]]). Let $(X,\tau)$ be a [[metric space]], and $S\subseteq X$. The _closure of_ $S$ is [[Closed Set|closed]].
